# stadvdb-mco2

Movie seat reservation web application with both transactional and analytical operations. 

## To run the program,

1. Ensure that you have `uv` installed.
2. Run `uv sync`.
3. `docker-compose down -v`
4. `docker-compose up --build`
5. Access the frontend app
